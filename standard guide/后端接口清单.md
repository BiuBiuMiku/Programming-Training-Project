## 三种取参方式代码示例

### 1. POST body JSON

```cpp
auto json = req->getJsonObject();
if (!json) { /* 返回 40001 参数错误 */ }

std::string phone = (*json)["phone"].asString();
std::string code  = (*json)["code"].asString();
std::string password = (*json)["password"].asString();
```

### 2. GET query 参数

```cpp
// 前端发: GET /api/trains?from=BJP&to=SHH&date=2026-07-10
std::string from = req->getParameter("from");   // "BJP"
std::string to   = req->getParameter("to");     // "SHH"
std::string date = req->getParameter("date");   // "2026-07-10"
```

### 3. Header 取 Token

```cpp
std::string auth = req->getHeader("Authorization");
// auth = "Bearer eyJhbGci..."
// 截掉 "Bearer " 前缀得到 JWT
// 解析 payload 中的 role 字段判断权限
```

## 三条铁律

1. **响应包层** —— 所有接口返回 JSON 必须是 `{"code": 0, "message": "...", "data": ...}`。code 为 0 表示成功，非 0 前端一律拿 message 展示错误。

2. **字段名严格一致** —— 按照 API 规范文档里的命名，驼峰式，一个字母都不要差。
   - 正确：`trainNo`、`fromStation`、`departureTime`
   - 错误：`train_no`、`from_station`、`departure_time`

3. **Token 校验失败统一返回** ——
   ```json
   {"code": 40100, "message": "未登录或Token已过期", "data": null}
   ```

## 开发阶段验证码

`POST /api/auth/send-code` 收到请求后，开发阶段直接将验证码打印到控制台，或固定为 `123456`。前端注册页输入此验证码即可完成注册。
